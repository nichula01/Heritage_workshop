# Project Overview

## Project
Heritage workshop paper

## Current direction
Vision-language model based research for cultural heritage understanding.

## Goal
Build a solid workshop paper with:
- clear task definition
- direct dataset usage
- reproducible code
- strong figures
- paper draft materials

## Main folders
- `docs/` notes, plans, literature, ideas
- `paper/` abstract, outline, draft sections
- `figures/` paper figures
- `data/` dataset metadata and processing notes
- `notebooks/` exploration
- `src/` source code
- `scripts/` runnable scripts
- `outputs/` generated results
